# Portfolio-Builder

. Built a web-based application for creating and customizing professional portfolios with real-time previews,
. Implemented drag-and-drop functionality using React.js, reducing portfolio creation time by 40%,
. Designed scalable backend services and integrated MongoDB for data storage,
. Deployed the application on AWS EC2 and S3, ensuring fast load times and scalability,
. Tech Stack: React.js, Node.js, MongoDB, AWS.
